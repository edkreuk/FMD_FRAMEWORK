# FMD Branding Pack for Fabric Portal

This folder contains custom branding assets for Microsoft Fabric:

| File | Description |
|---|---|
| `fmd-logo.svg` | Master logo (512x512) — concentric ring design with FMD text and "Fabric Metadata-Driven Framework" subtitle |
| `fmd-cover.svg` | Master cover artwork (1920x1080) — includes embedded fmd-logo, title, subtitle, and medallion layer diagram |
| `fmd-logo-portal.png` | Fabric-ready logo (200x30, ≤10 KB) |
| `fmd-cover-portal.jpg` | Fabric-ready cover image (1920x160, ≤1 MB) |
| `fmd-cover-portal.png` | Fabric-ready cover image (1920x160, ≤1 MB) |
| `theme-color.txt` | Full color palette used across all branding assets |

## Recommended Fabric portal settings

| Setting | Value |
|---|---|
| Logo | `fmd-logo-portal.png` |
| Cover image | `fmd-cover-portal.jpg` (or `fmd-cover-portal.png`) |
| Theme color | `#0F766E` |

## Verified portal constraints

| File | Dimensions | Size |
|---|---|---|
| `fmd-cover-portal.jpg` | 1920x160 | 27.98 KB |
| `fmd-cover-portal.png` | 1920x160 | 61.55 KB |
| `fmd-logo-portal.png` | 200x30 | 2.21 KB |

## If SVG upload is not accepted

Export the SVG files to PNG with the same dimensions:

- `fmd-logo.svg` → `fmd-logo.png` (512x512)
- `fmd-cover.svg` → `fmd-cover.png` (1920x1080)

Any image editor that supports SVG export can be used (Figma, Inkscape, Illustrator, or web converters).
